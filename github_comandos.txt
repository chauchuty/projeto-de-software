Passos Iniciais para usar GitHub

[ 1 ] - Toda vez que abrir o projeto, ATUALIZE com o comando (git pull)
[ 2 ] - Prepare os arquivos para o commit (git add .)
[ 3 ] - Faça o commit (git commit -m "add: arquivo com instruções do github")
[ 4 ] - Suba para o repositório (git push)

Resumo comandos:
git config --global user.name "<name>"
git config --global user.email "<name>"
git clone <repository>
git pull
git add .
git commit -m "mensagem..."
git push
